# 文本编辑器通用操作技巧



##### 选中

双击 `鼠标左键` ：选中单词、数字等单个字符串

三击 `鼠标左键` ：选中整行（一般包括换行符）

> Windows 换行符：`CR` + `LF`（即 `\r\n` ）
> Linux 换行符：`LF`（即 `\n` ）

`Ctrl` + `A` ：选中全部

`鼠标左键` 单击开始位置 + `Shift` + `鼠标左键` 单击结束位置：选中指定区域

##### 编辑

`Ctrl` + `X` ：剪切

`Ctrl` + `C` ：复制

`Ctrl` + `V` ：粘贴

`Ctrl` + `Z` ：撤销

`Ctrl` + `Y` ：恢复（某些文本编辑器中为其他快捷键）

`Ctrl` + `F` ：查找

`Ctrl` + `/` ：单行注释（IDE 中）

##### 字体格式

`Ctrl` + `B` ：粗体

`Ctrl` + `I` ：斜体

`Ctrl` + `U` ：下划线

##### 段落格式

`Tab` ：向右缩进

`Shift` + `Tab` ：向左缩进（某些文本编辑器中为其他快捷键）
