# 论文作图技巧

### 数据图

目标：生成能够用于 LaTeX 的矢量图。

#### Workflow

环境：
- Python
  - matplotlib
- Codex
- inkscape
- PowerPoint

##### 1. AI 编写 matplotlib 代码生图，导出为 svg

设置 `"svg.fonttype": "none"` ，避免默认将文字导出为 `path`。

##### 2. 第三方库（如 Inkscape ）将 svg 转为 emf

这一步可以使用 Inkscape 的应用程序手动完成，但使用 Python 代码调用命令行工具更方便。

1.x 版本：

```python
subprocess.call(
    f'inkscape "{svg_path}" --export-type=emf --export-filename="{emf_path}"',
    shell=True,
)
```

0.9x 版本：

```python
subprocess.call(f'inkscape "{svg_path}" -M "{emf_path}"', shell=True)
```

在此之前，需自选版本安装 Inkscape ：[Inkscape Release](https://inkscape.org/release)

例如，下载并解压 [Inkscape v1.4.3 compressed(7z)](https://inkscape.org/release/inkscape-1.4.3/windows/64-bit/compressed-7z/dl/) ，将 bin 目录加入环境变量。

验证是否可用：

```shell
inkscape --version
```

##### 3. PPT 编辑 emf

将 emf 文件导入 PPT ，右键-组合-解除组合，重复上述步骤直至每个元素都单独存在。
